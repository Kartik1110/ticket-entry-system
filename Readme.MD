- [Vercel](https://vercel.com) - Hosting

## Setup this project locally

### Client Setup

- Run 'cd client'
- Add .env file
- Copy .env.example to .env
- Add VITE_API_BASE_URL - URL for backend
- Run 'npm install'
- Run 'npm run dev'

### Server Setup

- Run 'cd server'
- Add .env file
- Copy .env.example to .env
- Add DATABASE_URL - URL for mongodb server
- Run 'npm install'
- Run 'npm run dev'
